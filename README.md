Synopsis
Hearts is a felxible playing cards API written in TypeScript.

Motivation


Installation
npm install hearts --save

dist/hearts.js to your project.

API Reference

let deck = new Deck(1);

deck.shuffle();
deck.draw();
deck.cards;
